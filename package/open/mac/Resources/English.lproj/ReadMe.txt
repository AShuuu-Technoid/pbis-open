PowerBroker Identity Services - Open Edition
--------

To join an Active Directory domain using a graphical utility, run:

        /Applications/PBIS Utilities/Domain Join

To join an Active Directory domain using a command-line interface, run:

        /opt/pbis/bin/domainjoin-cli

Documentation for this software is available at the following location(s):

	https://www.powerbrokeropen.org/resources/
